## Task Manager Project

A simple web-based task manager application built with HTML, CSS, and JavaScript.

## How to use

visit http://www.taskmanager.dpdns.org/

## Features

- **Task Management**: Create, edit, and delete tasks.
- **All Tasks View**: View all your tasks in one place.
- **Reports**: Generate and view reports on your tasks.
- **Timer**: Use a built-in timer for productivity.
